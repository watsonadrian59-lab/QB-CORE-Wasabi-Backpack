wasabi_backpack
This resource was created as a free script for backpacks using QBInventory

Features:

0.0 ms Usage
Perisistent backpack prop added to back when in inventory
Customizable item name and storage parameters
Compatibility for QBCore ONLY.
Installation
Download this script
Add wasabi_backpack to [standalone]
Add backpack image to inventory images (found in wasabi_backpack/_inventory_images/backpack.png)
ensure wasabi_backpack after `ox_lib' but before QBInventory.
Dependencies
QBCore
QBInventory
Oxlib
